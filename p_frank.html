Frank "Weekie" Podraza

My Name is Frank Podraza, I go by Weekie on YouTube, Instagram, Twitter, and pretty much any social media I'm on. I created the Nerd Herd Network, I run the Nerd Herd YouTube Channel and this website.

I like to make games. I developed Poke My Balls back in 2016, the CSGO Prop in 2019, and I'm currently working on Launch My Dog. I am a student at the University of California Santa Cruz. I am studying computer science.

Poke My Balls

Poke My Balls started as a project called Tap2Kill. The concept was the same. It's a simple game where different colored balls fall and you have to tap all of one color, but none of the other colors. My Father came up with the name one day and it stuck. I used this project as a way to start learning programming, and the Unity engine. I have come back to it over the years to implement concepts I have learned in class. Most recently, I used a class structure and polymorphism to make the game modes load as if they were game cartridges being put into a game console. There is one system that handles loading, and storing data (The "console"), and another that handles all the behaviors of the game (The "cartridge"). Check it out!

Launch My Dog

Launch My Dog is a game where you launch a hot dog into a bun. I almost finished this game back in 2018, but felt that it wasn't good enough to release. It didn't feel fun. I came back to it early 2020 and have been slowly working on it since. The goal is a tight small game that I can release on Mobile. There's some fun programming in here like trajectory prediction, even through different mediums. I have learned a lot about optimization with this project. The original had all the levels loaded simultaneously, which was slow. Now it loads each level additively, which is considerably faster, and allows me to push other aspects of the game. Check it out!
